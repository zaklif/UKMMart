# UKMart

A web-based e-commerce system developed using PHP and MySQL.

## Features

- User authentication
- Product management
- Shopping cart system
- Product search functionality
- Order management
- Admin dashboard

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Project Overview

UKMart is an e-commerce platform designed to provide users with an online shopping experience while allowing administrators to manage products, users, and orders efficiently.
